SDSU Anatomy Lab Simulator


**Purpose**

- This application was designed by me to simulate the Practical environment in the SDSU Anatomy Program
- When the simulation is run, the users can select a folder with questions that are similar to what they will see on both practicals and quizzes 


**Future Goals and Features**
- multiple deck/question pack selection
- randomized selection of questions
- Android/iOS apps

**Why?**

- As a pre med student, I felt largely unprepared for my first quiz and practical. I realized I could utilize my previous experience in software to create a tool that truly simulates the practical and quiz environment.
